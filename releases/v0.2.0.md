# v0.2.0

**Released:** 2026-02-28

Changes since v0.1.0 (`1514e86`..`078fd87`).

## Features

- **Omarchy theme auto-sync** (PR #38 by @LuccaRomanelli) — When running on the [Omarchy](https://omarchy.dev) desktop environment, FrankMD detects the terminal theme from `~/.config/omarchy/current/theme/alacritty.toml` and adds it as a selectable "Omarchy" theme. Polls for changes so switching your terminal theme updates FrankMD in real time.

- **New Folder in context menu** (PR #39 by @LuccaRomanelli) — Right-click a folder in the explorer to create a new subfolder directly from the context menu.

## Bug Fixes

- **Drag-and-drop refactored to Turbo Streams** (PR #23 by @rafaself) — File and folder moves now use a single Turbo Stream request instead of JSON + manual DOM manipulation. Removed redundant guards and consolidated validation.

- **Table hint behavior** (PR #36 by @rafaself) — Fixed table hint visibility management: hints no longer flicker, appear only when the cursor is inside a markdown table, and clean up properly when navigating away.

- **Ollama integration** (PR #34 by @murilo-develop) — Fixed `ModelNotFoundError` when using Ollama and corrected the API base URL handling so custom `ollama_api_base` values work properly.

- **Hugo post creation race condition** — New Hugo posts now open reliably by passing the created file path via a response header, avoiding a Turbo 8 DOM race. File finder results are also sorted by recency.

- **Scroll jitter with Preview open** — Fixed scroll jumping during text selection when the preview panel is visible.

## Security

- Updated `faraday`, `nokogiri`, and `rack` for security advisories.
